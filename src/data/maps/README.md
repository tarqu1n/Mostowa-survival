# Map data

Authored map files (`*.map.json`), the world layout (`world.json`), and the generated manifest
(`manifest.json`) that the in-repo map editor (plan 014, `npm run editor` once step 4 lands)
produces and the game loads. Edit maps and world placements through the editor UI only — never
hand-edit a map's cell arrays or `world.json`'s placements, and never hand-edit `manifest.json` at
all (it's regenerated by the editor's save middleware on every map/world save; see
`src/systems/mapFormat.ts` and `src/systems/worldLayout.ts` for the authoritative schemas and the
single parsing choke point every file here goes through).
